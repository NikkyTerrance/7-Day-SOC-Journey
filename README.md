# 7-Day SOC Analyst Learning Path

## About This Repository

This repository documents my journey through the **TryHackMe SOC Level 1 Learning Path** as I build practical skills and hands-on experience in Security Operations.

## Background

I recently graduated with **First Class Honours in BSc Cyber Security and Forensic Computing** and am actively pursuing SOC Level 1 Analyst positions. To bridge the gap between academic knowledge and industry requirements, I'm completing this intensive learning path to gain practical experience with the tools, techniques, and workflows used by SOC analysts in real-world environments.

## Learning Objectives

Through this 7-day challenge, I aim to:

- Develop proficiency in security monitoring and threat detection
- Gain hands-on experience with industry-standard SIEM tools
- Understand incident response procedures and workflows
- Practice analyzing security alerts and investigating potential threats
- Build a portfolio of documented case studies and solutions

## Repository Structure

This repository contains:

- **Daily progress logs** documenting key learnings and challenges
- **Room write-ups** with detailed explanations of concepts and techniques
- **Notes and resources** compiled throughout the learning path
- **Practical exercises** and their solutions

## Why SOC Analysis?

Security Operations Centers are at the frontline of cybersecurity defense. As organizations face increasingly sophisticated threats, skilled SOC analysts are essential for detecting, investigating, and responding to security incidents. This learning path allows me to develop the practical skills that complement my academic foundation.
